# S8.Phase1.08: Named-lead drafted welcome message

**Status:** Shipped (2026-05-22)
**Assignee:** @dev
**Epic:** S8.Phase1
**Created:** 2026-05-18

## User narrative

A new client should not receive a generic "Welcome to Powell Litigation" message. The first message should be signed by the lawyer who will be their actual point of contact, with that lawyer's name and title. Phase 1 ships a drafted welcome template that merges in the matter's lead lawyer (snapshotted from firm config at take time). The draft is reviewed and edited by the lawyer before send. Auto-send is explicitly avoided so LSO Rule 4.2-1 protection holds.

## Acceptance criteria

- [ ] Pure helper `buildWelcomeDraft(matter, leadLawyer): { html: string, plain_text: string }` produces a templated welcome message. Uses fixed approved snippets only. Includes the lead lawyer's `display_name`, `title`, and the firm name. Surfaces the matter type label and a single next-step sentence (matter-stage appropriate). No outcome promises, no specialist or expert language, no unverifiable superlatives.
- [ ] Take-action handler (Story 3) calls `buildWelcomeDraft` after creating the matter and stores the result on `client_matters.welcome_draft_html`.
- [ ] New page surface: a banner panel on `/portal/[firmId]/clients/[matterId]` (visible only when `welcome_draft_sent_at IS NULL`). Banner shows the drafted message, an editable textarea (pre-filled), and a "Send invite" button.
- [ ] New API route `PUT /api/portal/[firmId]/matters/[matterId]/welcome-draft`: saves an edited draft without sending.
- [ ] New API route `POST /api/portal/[firmId]/matters/[matterId]/welcome-draft`: sends the current draft. On send: (a) `welcome_draft_sent_at` is set to `now()`, (b) the magic-link invite is fired via Story 1's invite endpoint, (c) the sent body is written as the first message on the matter at `channel_type='client'`, (d) `matter_stage` transitions from `intake` to `retainer_pending`.
- [ ] Send is idempotent on `matter_id`: a second POST after `welcome_draft_sent_at` is set returns 409 with the existing send timestamp.
- [ ] Approved-snippets pool lives in `src/lib/welcome-snippets.ts` as a typed map. Adding a new matter-type variant means adding a new key to the map. Operator-curated; per-firm overrides deferred to Phase 2.
